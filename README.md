#MISKO

#El reto
Diseñar un sistema capaz de evaluar la calidad de las respuestas proporcionadas
por modelos de lenguaje, permitiendo identificar respuestas incorrectas,
alucinaciones, sesgadas o inapropiadas. El propósito es mejorar la confianza y
eficaciade las interacciones entre humanos y sistemas de inteligencia artificial
