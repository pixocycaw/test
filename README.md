# MISKO

## El reto

Diseñar un sistema capaz de evaluar la calidad de las respuestas proporcionadas por modelos de lenguaje, permitiendo identificar respuestas incorrectas, alucinaciones, sesgadas o inapropiadas. El propósito es mejorar la confianza y eficacia de las interacciones entre humanos y sistemas de inteligencia artificial.

## The challenge

Design a system capable of evaluating the quality of responses provided by language models, allowing the identification of incorrect answers, hallucinations, biased or inappropriate responses. The purpose is to improve the confidence and effectiveness of interactions between humans and artificial intelligence systems.
